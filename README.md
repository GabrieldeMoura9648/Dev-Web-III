## 📖 Sobre o Projeto

Este repositório contém exercícios e implementações desenvolvidas na disciplina de Desenvolvimento Web III, com foco na criação de aplicações web no lado do servidor utilizando Node.js.

O projeto demonstra conceitos fundamentais de back-end, como manipulação de requisições HTTP, definição de rotas e envio de diferentes tipos de resposta.

---

## ⚙️ Tecnologias Utilizadas

* JavaScript
* Node.js
* HTML
* HTTP

---

## 🚀 Como Executar

```bash
git clone https://github.com/GabrieldeMoura9648/TPII.git
cd DevWebIII
node server.js
```

Acesse no navegador:

```
http://localhost:3000
```

---

## 🌐 Rotas Disponíveis

* `/` → Página inicial
* `/html` → Retorna conteúdo HTML
* `/json` → Retorna dados em JSON
* `/jpeg` → Retorna imagem
* `/pdf` → Retorna arquivo PDF

---

## 🎯 Objetivo

Aplicar na prática conceitos essenciais de desenvolvimento back-end:

* Criação de servidor HTTP
* Manipulação de rotas
* Uso de diferentes tipos de resposta (HTML, JSON, arquivos)
* Organização básica de um projeto web

---

## 📌 Considerações

Este projeto serve como base para evolução em desenvolvimento back-end, incluindo uso de APIs e frameworks como Express.js.
